Req1.资金池信息展示
目标/价值
提供资金池余量信息、业务模式介绍、平台成交情况，金融业务核心要素
Req1.1 资金紧张度：平台总可用资金情况展示
剩余总额度：总可用额度-已用额度
总可用额度：运营提供，后台手工录入
已用额度：邦帮贷系统中的已确认、未结清的资方融资单剩余本金合计
计算频次：每整点一次
若计算后数值≤0，则显示“暂无可用资金，可联系我们预约”
数据传输：服务端处理数据展示位置后，传输至前端，不直接传输数字
显示：
数值：
四舍五入精确至万
曲线图
不定积分曲线区域，红绿渐变，高为绿色，低为红色
横坐标轴：天数，1-30天，中间数值不显示
纵坐标轴：剩余总额度
刻度最大值计算方式：取30日内30条数据的最大值，只入不舍至千万位
注：若当日数据在更新后为最大值，纵坐标刻度也需要更新
1-29天历史数据，展示该日24：00计算的结果，需根据该日该时的“总可用额度”、“已用额度”计算，不根据“总可用额度”的改变而改变
第30天数据展示每隔1小时计算的结果
资金紧张度：定性展示
计算公式：已用额度/总可用额度
已用额度：邦帮贷系统中的已确认、未结清的资方融资单剩余本金合计
总可用额度：运营提供，后台手工录入
显示：
0%≤x＜25%：观望
25%≤x＜50%：平稳
50%≤x＜75%：火热
75%≤x：买买买！
计算频次: 每个整点计算
